# Pixiv令牌

#### 介绍

python获取pixiv令牌

**使用说明：**

下载本仓库ZIP后

解压

在解压文件的目录pixiv文件夹内启用终端

![输入图片说明](https://foruda.gitee.com/images/1717261221424272330/8ad05889_11990909.png "37157DFC40EBBE7A0B5600F9421D3889.png")

**然后执行**

~~~
python pixiv_auth.py login
~~~

**登录账号后，复制返回的链接到终端回车即可**
